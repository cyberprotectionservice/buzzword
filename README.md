> Web-app for corpus linguistics

## Install buzzword

```bash
pip install buzzword
```

First, configure a `.env` file from `.env.example`, a `corpora.json` file from `corpora.json.example`, and then do:

```bash
buzzword
# or
python -m buzzword
```
